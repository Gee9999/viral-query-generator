# PROMPT BUILDER V2.0 PRO

## Setup & Running
This project uses Vite.

### Important: Port Configuration
To avoid conflicts with other local projects (like Proto Predict), this app is configured to run on port **8888**.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to:
   **http://localhost:8888**

## Features
- **✨ Dual Mode Output**: Generates both your raw prompt and a professionally "AI-Enhanced" version with typewriter animation.
- **📚 Prompt Memory**: Integrated Library to save/load/delete prompts instantly in the browser. Includes a "Backup All" feature.

- **⚡ Quick Templates**: Pre-loaded templates for Web Apps, Data Scripts, and Marketing.
- **💾 Full Persistence**: Save projects to disk (`.json`) or browser memory.
- **🚀 Production Ready**: Optimized for Vercel deployment.

## Deployment (Vercel)
1. Run `npx vercel` in this folder.
2. Follow the prompts (use default settings).
3. Copy the production link to share!
